จากนั้น
1. Clone โปรแกรมโดยใช้คำสั่ง https://github.com/kimmattic/password_random.git
2. จากนั้น Run "python main.py"
3. จากนั้นใส่จำนวน Password ที่ต้องการให้สุ่มครับ
